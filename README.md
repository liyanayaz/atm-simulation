# ATM Simulation

A simple ATM simulation built with Object-Oriented Programming concepts.
Implemented in three different languages: C++, Python, and Java.

## Features
- Balance inquiry
- Deposit
- Withdrawal (with insufficient funds validation)
- Card return

## Languages
| File | Language |
|------|----------|
| atm_simulation.cpp | C++ |
| atm_simulation.py | Python |
| ATMSimulation.java | Java |

## How to Run

**C++**
```bash
g++ atm_simulation.cpp -o atm
./atm
```

**Python**
```bash
python3 atm_simulation.py
```

**Java**
```bash
javac ATMSimulation.java
java ATMSimulation
```
